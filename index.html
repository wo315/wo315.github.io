The software development process

我就是我，不一样的烟火。

  1. We have now gone though the various phases in the process of writing a software. These phases can be summarised as follows:
  2. What(Analysis)
  3. How(Design)
  4. Do it(Implementation)
  5. Test(Testing and Debugging)
  6. Use(Operation or Deployment)
  7. Maintain(Refinement)

A recommended way of writing programs is the procedure we have followed in creating the backup scripu: Do the analysis and design. Startimplementing with a simple version. Test and debug it. Use it to ensure that it works as expected. Now, add any features that you want and continue to repeat the Do-It-Test-Use as many times as required
